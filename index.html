Ethereum Meeting Rooms Booking

This is a simple example of decentralized application built on the Ethereum platform. Here you can find the tutorial that explains the DApp in all its parts, starting with the setting of the development tools.

Bookings will be stored in the Ganache test blockchain. Obviously it is not possible to book a time slot if it is already booked. You can not book a time slot in a meeting room if you have already booked the same slot in another room. Your booked slots are yellow; the slots booked by other users are red.
